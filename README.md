FastCar - Sistema de Locação de Carros em Python

FastCar é um protótipo de sistema de locadora de carros feito em Python.  
Ele simula toda a lógica básica de uma locadora: escolha de carro, dias de aluguel, seleção de seguro e cálculo do valor total.  
O objetivo do projeto é praticar lógica de programação, estruturas de dados e interação com o usuário.

 Funcionalidades:
 - Menu interativo com opções "Alugar carro" ou "Sair" 
- Listagem de carros com preço e tipo (Manual / Automático) 
- Seleção de carro pelo número da lista e conversão para índice
- Solicitação de quantidade de dias de locação
- Seleção de seguro usando dicionário, acessando dinamicamente as chaves
- Cálculo do valor base
- Cálculo do valor do seguro 
- Soma do valor base e do seguro para obter o total da locação  
- Exibição de resumo detalhado da locação

- Conceitos usados:
- - Listas e dicionários  
- Loops for e enumerate()  
- input() para interação com o usuário  
- Conversão de tipos com int()  
- keys() + list() para acessar chaves de dicionário dinamicamente  
- Condicionais if/elif para controlar o fluxo do programa  
- Operações matemáticas básicas (* e +)  
- F-strings para exibir informações formatadas

- Conclusão:
FastCar é um projeto simples, mas completo, que demonstra domínio de lógica, estruturas de dados e interação com o usuário.
Serve como base para projetos maiores, integrando banco de dados, interface gráfica e funcionalidades reais de locadora.
